Sioyek を使うためのかんたんガイド

ちゃんとしたドキュメントは Sioyek Documentation にある．cmd などは各自で環境にあったものに読みかえる．

File Open

  • o : ファイルを開く
  • O : 最近開いたファイルを開く
  • cmd+o : ディレクトリからファイルを開く（ sioyek 内の browser を利用）
  • cmd+O : 今いるファイルのディレクトリからファイルを開く（ sioyek 内の browser を利用）
  • cmd+t : 今開いているファイルを別ウィンドウに複製する

Basic Movement

  • : (colon) : help を表示
  • 42gg : pdf の 42 ページに移動
  • t : 目次 / toc を表示して移動
  • F8 : pdf の背景色を変更する（ダークモード）

Bookmark, Highlight

  • b : （選択範囲を設定して）ブックマークを設定する，色はつかない
  • gb : ブックマークを検索する
  • gB : 検索可能なすべての pdf からブックマークを検索する
  • ha : （選択範囲を設定して）名前 a のハイライトを設定する
  • gh : ハイライトを検索する
  • gH : 検索可能なすべての pdf からハイライトを検索する
